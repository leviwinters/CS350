# CS350

This project was about building a thermostat system using a Raspberry Pi. It reads temperature data, lets the user adjust a set temperature with buttons, and shows heating or cooling status with LEDs. It also simulates sending data to a server using UART.

I think I did well with getting the main functionality working and organizing the logic using a state machine to handle different system states.
One thing I could improve is debugging. I struggled at times to get parts of the code working correctly, and it took some trial and error to fix issues.

For tools and resources, I mainly used class materials and online documentation. I would continue building on that and try to use more detailed technical references in the future.

The main skills I gained that will transfer to other projects are working with hardware inputs and outputs, using state machines, and breaking problems into smaller parts to solve them.
